Text Compression Program

I learnt all about compression in the course that I took and am currently teaching, CS 240. It is an algorithms and data structures course. I wanted to see how compression algorithms are used in the real world. A lot of programs use some variation of Lempel-Ziv which I have coded. But, I decided to look more into Burrows-Wheeler Transform (BWT). I learnt that BWT normally is used with the Move To Front heuristic, Run Length Encoding (preferablly Zero RLE) and some form of entropy encoding. Since I was teaching the course as an ISA (Instructional Support Assistant) I decided to try a code out. This application was the result of that.
